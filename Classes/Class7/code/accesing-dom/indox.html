Accesing Elements Without Changing Them!

How browsers should create a model of an HTML page!

JavaScript can interact with it to access and update the web page.

The model is stored in the browser's memory.

It consists of four main nodes!

  • Document Node
  • Element Node
  • Text Node
  • Attribute Node
